# Week 4 – RESTful API Integration

## Overview

This week focuses on integrating REST APIs into a React application.

## Features Implemented

### ✅ GET Request

- Fetch data from JSONPlaceholder API
- Use useEffect for lifecycle handling
- Store response using useState
- Render dynamically using map()

### ✅ POST Request

- Send new data to API
- Triggered by button click
- Demonstrates RESTful POST method

## Concepts Learned

- REST APIs
- HTTP Methods (GET, POST)
- Asynchronous JavaScript
- React Hooks
