# Week 10 – User Interaction (Search & Filtering)

## Objective

Enhance user interaction by implementing search functionality.

## Features Implemented

- Search input field
- Real-time product filtering
- Dynamic UI updates
- Conditional rendering (no results message)

## Explanation

A search feature was implemented to allow users to filter products based on their input.

The filtering is done in real-time using the `useState` hook to track user input and dynamically update the displayed products.

If no matching product is found, a message is displayed to inform the user.

## Outcome

The application is now interactive, allowing users to easily find products, improving usability and overall user experience.
