# Week 8 – React Router

## Objective
Implement multi-page navigation using React Router.

## Features Implemented

- React Router DOM
- Navigation Menu
- Home Page
- Products Page
- Product Details Page
- Dynamic URL Parameters

## Explanation

React Router was introduced to transform the application into a multi-page experience.

The application now contains three main pages:

Home – Landing page of the application  
Products – Displays all fetched products  
Product Details – Displays information about a selected product

Dynamic routing was implemented using URL parameters to display product details.

## Outcome

The application now behaves like a real-world React web application with proper page navigation and routing.
