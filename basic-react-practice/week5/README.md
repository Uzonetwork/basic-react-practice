# Week 5 – React Hooks

## Overview

This week focuses on understanding and implementing React Hooks in a practical React application.

---

## Hooks Implemented

### useState

- Managed search input state
- Managed product state inside custom hook

### useEffect

- Fetch products from DummyJSON API
- Auto-focus input field on component mount

### useRef

- Accessed DOM element directly
- Automatically focused search input on page load

### Custom Hook – useProducts

- Abstracted API fetching logic
- Reusable hook returning products and loading state

---

## Features Implemented

- Product fetching from API
- Search filtering functionality
- Loading state handling
- Clean separation of logic using custom hook

---

## Concepts Learned

- React Hooks architecture
- Side effects handling
- DOM manipulation in React
- Code reusability with custom hooks
